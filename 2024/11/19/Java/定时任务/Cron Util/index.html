基本使用

定义cron表达式的支持范围

支持定时任务的类型：

  1. CRON4J
  2. QUARTZ
  3. UNIX
  4. SPRING
  5. SPRING53

生成cron表达式

各方法对应cron表达式关系：

  1. always：表示*
  2. questionMark：表示?
  3. on：表示具体值
  4. between：表示-，例如，between(0,5)表示0-5
  5. and：表示,，例如，and(on(1),on(5))表示0,5
  6. every：表示/，例如，every(on(2),3)表示2/3
